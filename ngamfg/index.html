Next Generation Arms Manufacturing

Image last updated June 2017

Next Generation Arms(NGA) manufactures custom billet AR15 upper and lower receivers and extension tubes. All parts are original in-house designs made from high-quality materials.

This site is a simple, static site featuring a custom administration area written in PHP for the clients use. Utilizing a graphical editor, they are able to edit the content of the pages and upload images for use in their gallery.

The site is mobile friendly and conforms to current SEO practices.
