Panhandle Logworks

Image last updated June 2017

Panhandle Logworks offers custom and mobile milling services. They are equipped with a mobile milling unit that allows them to arrive on site and mill lumber to custom dimensions from logs that have been felled nearby. No need to transport the logs off site.

This site was created using WordPress and is maintained by the client. The theme and layout are custom and feature a drag and drop interface to make it easy for the website owner to update and customize their content.
